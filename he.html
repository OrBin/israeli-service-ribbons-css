How does it work?

All ribbons are represented by the same HTML structure

                        
    <figure class="komemiut war">
        <span class="outer"></span>
        <span class="inner"></span>
        <span class="outer"></span>
    </figure>

All ribbons use the base layout

                    
    .war {
        position: relative;
        overflow: hidden;
        display: flex;
        justify-content: center;
        width: 151px;
        height: 38px;
        box-shadow: 0 0 15px rgba(0, 0, 0, .2);
        border: 1px solid rgba(0, 0, 0, 0.5);
        border-radius: 3px;
        z-index: 1;
    }

Each ribbon overrides base style

                    
    .komemiut {
        background: #184ea4;
    }
    .komemiut .a1 {
        max-width: 7px;
        background: #92c8e2;
        border-right: 7px solid #f4f4f4;
        border-left: 7px solid #f4f4f4;
    }
    .komemiut .b1 {
        margin: 0 1.65rem;
        max-width: 7px;
        background: #ab3122;
        position: relative;
    }
